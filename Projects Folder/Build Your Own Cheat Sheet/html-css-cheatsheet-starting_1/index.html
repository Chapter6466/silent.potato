HTML Table Reference

Table Tags

Tag Name Description
<table> Table Defines a table
<th> Table Header Defines a header cell in a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a cell in a table
<thead> Table Head Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Footer Groups the footer content in a table
<caption> Table Caption Defines a table caption
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting
<col> Column Specifies column properties for each column within a <colgroup> element

Table Attributes

Attribute Name Description
colspan Column Span Specifies the number of columns a cell should span
rowspan Row Span Specifies the number of rows a cell should span
headers Header Cells Specifies one or more header cells a cell is related to
scope Header Cell Scope Specifies whether a header cell is a header for a column, row, or group of columns or rows
colgroup Column Group Specifies a group of one or more columns in a table for formatting
col Column Specifies column properties for each column within a <colgroup> element
border Border Specifies the width of the border around a table
cellpadding Cell Padding Specifies the space between the cell wall and the cell content
cellspacing Cell Spacing Specifies the space between cells
width Width Specifies the width of a table
height Height Specifies the height of a table
align Alignment Specifies
